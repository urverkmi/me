Window

UI/UX Max/MSP

A digital window designed to be a seamless workspace companion. The project leverages Max/MSP for its real-time audio processing capabilities to create responsive visual elements.

Background

Office workers often face an inherent disconnect between indoor environments and the world outside of their office.

Window is a dynamic digital installation that restores this link through real-time audio-visual feedback.

How it works

Max/MSP is a visual programming language for creating interactive multimedia applications.

The Max patch takes the time-domain audio input and applies fast fourier transform (FFT) to obtain the frequency-domain data, which determines the size of the rhythmic visuals.

Behind the scenes, there is a python script that calls the OpenWeatherMap API and sends the weather data to the patch, which parses the message and toggles the corresponding visual elements.


The window demonstrates how a simple integration of natural elements can enhance our indoor work environments.
