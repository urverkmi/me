Friction<

Embedded systems UI/UX Group project

Sponsored by Aurora Propulsion Technologies, our task is to design a robust test rig system for their plasma brake product.

My main role is to develop the user interface for providing real-time feedback of the test status, integrated with controls to start or pause a test session.

Developed as part of Aalto PDP (Product Development Project).




Responsibilty

I lead the data measurement team. We are developing a pipeline to gather the test data, compute the relevant metrics (i.e. velocity, acceleration, tether length), and display the data on a graphical user interface (GUI) with an animation created on the fly.




GUI

I started our development by splitting up the pipeline into independent segments. We started by creating the GUI with mock data, which will be replaced with real data once code is ready for integration.
